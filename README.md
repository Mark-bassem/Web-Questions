# Quiz HTML & CSS — Score & Leaderboard

Quiz game (RTL) built with **HTML, CSS, and vanilla JavaScript**.

## Features
- Quiz flow: **Intro → Quiz → Result → Leaderboard**
- Scoring with time bonus
- **Shared leaderboard** across devices via **Supabase**
- Fallback to `localStorage` if Supabase isn’t configured

## Project Structure
- `html-quiz-game/index.html`  
  UI + embedded quiz data (`window.QUIZ_DATA`)
- `html-quiz-game/styles.css`  
  Styling
- `html-quiz-game/app.js`  
  Game logic + leaderboard logic

## Setup (Supabase Shared Leaderboard)
> Because GitHub Pages hosts static files only, the shared leaderboard needs a backend/database.

### 1) Create table
Run in Supabase SQL Editor:

```sql
create table if not exists public.leaderboard (
  id bigint generated by default as identity primary key,
  name text not null,
  score integer not null,
  time bigint not null
);
```

### 2) Enable RLS + public policies (simple for demos)
```sql
alter table public.leaderboard enable row level security;

create policy "public_select_leaderboard"
on public.leaderboard
for select
using (true);

create policy "public_insert_leaderboard"
on public.leaderboard
for insert
with check (true);
```

### 3) Add Supabase keys in `html-quiz-game/app.js`
Open `html-quiz-game/app.js` and set:
- `SUPABASE_URL`  (Project Settings → API → Project URL)
- `SUPABASE_ANON_KEY` (Project Settings → API → anon public key)

### 4) Deploy
1. Push to GitHub
2. Enable **GitHub Pages**
3. Open the same link from different devices → leaderboard should be shared

## Notes
- Leaderboard shows **top 10** by `score` (desc), then `time` (asc).
- Insert-only approach: each save creates a new row.
